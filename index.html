About CocoaLana

CocoaLana is a social media management tool built for cross-border e-commerce businesses. It enables merchants to efficiently publish product images and content to multiple social media platforms from a single desktop application.

Our tool helps e-commerce sellers save time by automating the process of posting product photos, writing captions, managing hashtags, and tracking engagement analytics across all major social platforms.

Supported Platforms

CocoaLana integrates with the following social media platforms via their official APIs:

Instagram Facebook Pinterest X (Twitter) TikTok

How We Use Pinterest API

CocoaLana uses the Pinterest API v5 to provide the following features for our users:

Pin Creation

Users can create and publish Pins with product images directly from our application to their Pinterest boards.

Board Management

Users can view and select their Pinterest boards as publishing destinations for their product content.

Scheduled Publishing

Users can schedule Pins to be published at optimal times for maximum engagement.

Analytics

Users can view Pin performance metrics including impressions, saves, and clicks to track content effectiveness.

OAuth Authentication

CocoaLana uses Pinterest's official OAuth 2.0 authentication flow to securely connect user accounts. Users authorize our application through Pinterest's login page, and we store access tokens locally on the user's device. We never collect or store Pinterest login credentials.

Required OAuth scopes: pins:read, pins:write, boards:read, boards:write, user_accounts:read
